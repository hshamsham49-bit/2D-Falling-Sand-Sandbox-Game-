# 🎨 Sandbox 2D - Particle Simulation Game

A falling sand style sandbox game built with HTML5 Canvas and vanilla JavaScript.

## 🚀 How to Play

1. Open `index.html` in your browser
2. Select a material from the sidebar
3. Click and drag on the canvas to place particles
4. Watch them interact!

## 🧱 Materials

| Material | Type | Behavior |
|----------|------|----------|
| 🏖️ Sand | Powder | Falls down, piles up |
| 💧 Water | Liquid | Flows down and sideways |
| 🔥 Fire | Gas | Rises, burns wood/oil/plant |
| 🪨 Stone | Solid | Doesn't move |
| 🪵 Wood | Solid | Burns when touched by fire |
| 💨 Steam | Gas | Rises and disappears |
| 🛢️ Oil | Liquid | Flammable liquid |
| 🌱 Plant | Solid | Grows with water, burns |
| 🧪 Acid | Liquid | Dissolves most materials |
| 🧂 Salt | Powder | Dissolves in water |

## ⌨️ Controls

- **Mouse/Touch**: Draw particles
- **Space**: Pause/Resume
- **C**: Clear canvas
